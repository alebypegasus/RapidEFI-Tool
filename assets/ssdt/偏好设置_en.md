## RapidSSDT Preferences (Compiler Settings)

Default options are recommended for most users.

- **iasl Mode**: *Built-in (Recommended)* — Uses the integrated offline IASL compiler.
- **iasl Compiler Type**: *Modern iasl* — Compatible with modern ACPI specifications.
- **Delete .dsl source files**: *Yes* — Deletes intermediate .dsl files after generating .aml.
- **Force Compile**: *Yes* — Bypasses non-critical warnings during compilation.
- **Overwrite Target EFI**: *Yes* — Copies SSDTs directly to target EFI directory with backup.
